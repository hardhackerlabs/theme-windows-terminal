## Install

Copy `theme.json`'s content , paste to Windows terminal's settings.json `schemes` array,
and then change `profiles.colorScheme` to `Hardhacker`ã€‚

```json
...
"profiles": {
    "colorScheme": "Hardhacker",
    ...
},
"schemes": [
        {
            "name": "Hardhacker",
            ...
        }
]
```
